Arithmetic operators
    x+y // plus
    +x // unary plus 
    x-y // minus 
    -x // unary minus
    x*y // multiply 
    x/y // dividw 
    x%y // remainder (modulus) for integers 

Comparison operators
    x==y // equal
    x!=y // not equal
    x<y // less than
    x>y // greater than
    x<=y // less than or equal
    x>=y // greater than or equal
