
String, vector, Matrix and Orc, are user-defined types.
Type can be deduced from the initializer:
    auto b = true; // a bool
    auto ch = 'x'; // a char
    auto i = 123; // an int
    auto d = 1.2; // a double
    auto z = sqrt(y) // z has the type of sqrt(y)'s return.

x+=y // x = x+y
++x // increment: x = x+1 
x-=y // x = x-y
--x // decrement: x = x-1
x*=y // scaling: x = x*y
x/=y // scaling x = x/y
x%=y // x = x%y
